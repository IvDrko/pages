Lenguaje de programación: C++

1. Presentación del lenguaje

C++ es un lenguaje de programación de propósito general, desarrollado por Bjarne Stroustrup en 1979 como una extensión de C. Es un lenguaje de programación compilado que permite la programación orientada a objetos (OOP), además de soportar otras características como la programación funcional y genérica. C++ es ampliamente utilizado para desarrollar aplicaciones de alto rendimiento como videojuegos, sistemas operativos, software embebido y aplicaciones de tiempo real.
