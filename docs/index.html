Lenguaje de programación: C++

1. Presentación del lenguaje

C++ es un lenguaje de programación de propósito general, desarrollado por Bjarne Stroustrup en 1979 como una extensión de C. Es un lenguaje de programación compilado que permite la programación orientada a objetos (OOP), además de soportar otras características como la programación funcional y genérica. C++ es ampliamente utilizado para desarrollar aplicaciones de alto rendimiento como videojuegos, sistemas operativos, software embebido y aplicaciones de tiempo real.

OnlineGDB es otro compilador en línea para C++ y otros lenguajes. Ofrece un entorno interactivo con soporte para depuración, lo que lo hace útil para proyectos más grandes.

Declaración de variables:

  • int - Para enteros.
  • float - Para números decimales de precisión simple.
  • double - Para números decimales de precisión doble.
  • char - Para caracteres.
  • string - Para cadenas de texto.
  • bool - Para valores booleanos (true o false).

Operaciones en C++:

  • Operaciones aritméticas: suma, resta, multiplicación, división, módulo
  • Operaciones relacionales: Igualdad (==), Desigualdad (!=), Mayor que (>), Menor que (<), Mayor o igual (>=), Menor o igual (<=)
  • Operaciones lógicas: AND lógico (&&), OR lógico (||), NOT lógico (!)

Condicionales en C++:

Las estructuras condicionales permiten ejecutar diferentes bloques de código dependiendo de una condición booleana. Se usan principalmente las instrucciones if, else if y else.

Ejemplo:

int edad = 18;
    if (edad >= 18) {
        cout << "Eres mayor de edad." << endl;
    } else {
        cout << "Eres menor de edad." << endl;
    }

Bucles en C++:

Los bucles en C++ permiten ejecutar un bloque de código varias veces. Los tipos de bucles más comunes son for, while y do-while.

Ejemplo de bucle for:

for (int i = 0; i < 5; i++) {
        cout << "i = " << i << endl;
    }

Ejemplo de bucle while:

int i = 0;
    while (i < 5) {
        cout << "i = " << i << endl;
        i++;
    }

Ejemplo de bucle do-while:

int i = 0;
    do {
        cout << "i = " << i << endl;
        i++;
    } while (i < 5);
